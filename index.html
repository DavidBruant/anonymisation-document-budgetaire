Anonymisation des fichiers <DocumentBudgetaire> du projet Actes Budgétaires

Les fichiers <DocumentBudgetaire> contiennent des noms de personne physique. http://odm-budgetaire.org/ Spécifiquement les éléments <CONCOURS> dans les Annexes dont l'élément CodNatJurBenef a pour valeur P3 http://odm-budgetaire.org/doc-schema/CommunAnnexe_xsd_Complex_Type_ATCodNatJurBenef.html#ATCodNatJurBenef_V

Cette page web permet d'anonymiser ces fichiers.
